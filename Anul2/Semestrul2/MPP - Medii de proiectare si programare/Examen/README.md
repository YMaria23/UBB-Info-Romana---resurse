# Informatii examen

Examenul este structurat in urmatorul mod (adaptat anului 2026):
- parte teoretica - aprox. 15 min
    - contine 2 exercitii in care trebuie explicate diverse concepte predate pe parcursul semestrului + exemplificare
    - un exercitiu de tip "ce face codul" in JavaScript
- parte practica - aprox. 2 ore
    - in general este vorba de un joc ce poate avea unul sau mai multi jucatori (in 2026 s-a mers pe varianta de mai multi jucatori)
    - cerintele au fost in 2026 in urmatorul fel:
        - inceput de joc (logare cu porecla + afisarea mesajului de "jucatori insuficienti" SAU "jocul poate incepe")
        - alegere configuratie / litera / etc.
        - mutare
        - finalizare joc (unde, in general, se includea si clasamentul jocurilor finalizate)
        - 2 cerinte de serivii REST care se puteau realiza **FARA A IMPLEMENTA JOCUL** (unul dintre el trebuia testat in Postman, ARC, etc., iar pentru celalalt era nevoie de client Web)
